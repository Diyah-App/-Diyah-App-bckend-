---

## 🌟 الفكرة والأهداف (The Concept)
مشروع "ميثاق الدية العشائرية" هو حل تقني يهدف إلى تحويل السجلات الورقية والتقليدية لإدارة الديات إلى نظام رقمي ذكي. يتيح للنظام تتبع المساهمات المالية، وإدارة شؤون الأعضاء، وضمان وصول المستحقات بشكل عادل ومنظم لجميع أفراد العشيرة (عشائر البو حمدان).

### المميزات الرئيسية:
- **إدارة الأعضاء**: تسجيل الوجهاء والأعضاء وتصنيفهم.
- **تتبع الديات**: إضافة ديات جديدة مع تحديد المبالغ، تواريخ الوقائع، والمشمولين بالدفع.
- **نظام محاسبي ذكي**: حساب حصة كل فرد تلقائياً بناءً على عدد المشمولين في تاريخ وقوع الدية.
- **إحصائيات مباشرة**: لوحة تحكم تعرض إجمالي المبالغ، المسددين، وغير المسددين.
- **واجهة مستخدم متجاوبة**: تصميم عصري يعمل بكفاءة على الهواتف الذكية وأجهزة الكمبيوتر (Web & Mobile).
- **نظام صلاحيات**: تسجيل دخول آمن للإدارة للتحكم في البيانات.

---

## 🛠 الأدوات والتقنيات (Tech Stack)

### الباك اند (Backend):
- **اللغة**: Python 3.x
- **الإطار البرمجي**: Flask (REST API)
- **قاعدة البيانات**: SQLite (خفيفة وسريعة)
- **التوثيق (Authentication)**: JWT (JSON Web Tokens)
- **الخادم**: WSGI / Gunicorn (للإنتاج)

### الفرونت اند (Frontend):
- **الإطار البرمجي**: Flutter (لضمان عمله على كافة المنصات)
- **اللغة**: Dart
- **التصميم**: Vanilla CSS/Flutter Material Design مع دعم كامل للـ Responsiveness.
- **إدارة الحالة**: StatefulWidgets مع مزامنة الـ API.

---

## 📂 هيكلية المشروع (Project Structure)
ينقسم المشروع إلى قسمين رئيسيين:
1. `backend/`: يحتوي على ملفات الخادم، المسارات (Routes)، وقاعدة البيانات.
2. `frontend/`: يحتوي على كود تطبيق فلاتر، واجهة الويب، ومنطق الأعمال.

---

## 📱 صفحات التطبيق (App Pages)
1. **شاشة البداية (Splash Screen)**: انيميشن تراثي يمثل هوية العشيرة (السيف والقلم).
2. **اللوحة الرئيسية (Dashboard)**: تعرض ميثاق الدية والبنود القانونية والاجتماعية.
3. **سجل الديات**: قائمة بجميع الديات (النشطة والمنتهية) مع فلاتر ذكية.
4. **شاشة الأعضاء والوجهاء**: عرض بيانات الأفراد وحالاتهم المالية.
5. **تفاصيل الدية**: عرض المشمولين، المبالغ المسددة، وتفاصيل الواقعة.
6. **ملف العضو الشخصي**: عرض تاريخ الديات الخاص بالعضو، المبالغ المترتبة عليه، والديات القديمة غير المشمول بها.

## 🚀 طريقة التشغيل (Setup & Installation)

### تشغيل الباك اند:
```bash
cd backend
python -m venv venv
source venv/bin/activate  # على ويندوز استخدم venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### تشغيل الفرونت اند:
```bash
cd frontend
flutter pub get
flutter run -d chrome  # للتشغيل على المتصفح
```
